Pasos

Se clona el archivo con git clone https://github.com/medmunval/prueba-git.git

Se crea el archivo .gitignore y en su interior se escribe *.jpg, luego se aplica git add .git ignore, luego git commit -m "Se creo el archivo .gitignore para omitir las imagenes con exeension jpg"

Se modifico el archivo introduccion.html, luego se aplica git add introduccion .html y git commit -m "Se modifico introduccion.html"

Se modifico el archivo marco_teorico.html, luego se aplica git add marco_teorico .html y git commit -m "Se modifico marco_teorico.html"

Se modifico el archivo desarrollo.html, luego se aplica git add desarrollo.html y git commit -m "Se modifico desarrollo.html"

Se modifico el archivo referencias.html, luego se aplica git add referencias .html y git commit -m "Se modifico referencias.html"

Se creo la rama diseno a traves del comando git brench diseno

Se ingreso a la rama diseno a traves del comando git checkout diseno

Se modifico el archivo estilo.css, se aplico el comando git add css, se aplico el comando git commit -m "Se modifico el color de fondo de la pagina"

Se modifico el archivo estilo.css, se aplico el comando git add css, se aplico el comando git commit -m "Se modifico el color de fondo de blockquote en el archivo css"

Se vuelve a la rama master con git checkout master

Se mezclan las ramas con git merge diseno

Se crea el archivo pasos.html

Se aplica git add pasos.html, luego git commit -m "Se creo el archivo pasos.html y se modifico instrucciones.html"

Se sube el archivo con git push origin master
